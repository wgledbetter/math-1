15 /* Template class for the combine step of map_rect which implements
16  * the CombineF concept. The concept requires that
17  *
18  * - A nullary constructor creates a null combiner (used on the
19  * children)
20  *
21  * - A constructor which takes the shared and job-specific parameters
22  * (used on the root/main process)
23  *
24  * - Provides an operator() which takes as two arguments: (i) the
25  * function outputs as a ragged matrix and (ii) as second argument
26  * the output sizes of each function evaluation.
27  *
28  * This functor inserts the concatenated outputs of all reduce
29  * operations into the autodiff stack. The concatenated results are
30  * stored in a double only matrix and is ragged according to the
31  * output sizes of each job.
32  *
33  * @tparam F type of user functor
34  * @tparam T_shared_param type of shared parameters
35  * @tparam T_job_param type of job specific parameters
36  */
